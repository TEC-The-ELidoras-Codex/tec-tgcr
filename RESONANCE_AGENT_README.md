## Governance & Sacred Guardrails

### Non-Negotiables

1. **Consent-first**: Every copyrighted track requires opt-in. Metadata-only by default; full embedding/sampling only with explicit permission.

2. **Provenance-always**: Every model output must include a card listing sources, authors, and confidence scores. Never present AI-generated content as original by an artist.

3. **Artist sovereignty**: AI suggestions are recommendations. Final creative control resides with humans.

4. **Revenue-sharing**: Default to micro-payments or revenue splits for any commercial derivative. Enforce via API (model refuses to output unless rights terms are met).

5. **Cultural respect**: Special care for marginalized creators (hip-hop artists, street poets, independent acts). Consent + opt-in defaults always.

6. **Transparency + audit logs**: Store model training logs, dataset manifests, and consent records. Make them available to artists and a transparency board.

7. **Advisory board**: Musicians, music lawyers, ethicists. They review contentious generations and flag cultural misuse.

### Data Retention & Deletion

- Artists can revoke consent anytime.
- Upon revocation: model re-trains without that artist's data; existing outputs remain but marked as "legacy" (no new generations using that artist).
- Consent records preserved for legal audit (but artist data deleted from embeddings).

## Tech Stack (Minimal Viable)

### Infrastructure

| Layer | Choice | Why |
|-------|--------|-----|
| Data store | S3 (audio/assets) + Postgres (metadata) | Scalable, auditable, standard |
| Embeddings | Vector DB: Weaviate or FAISS | Fast similarity search; can run locally for privacy |
| Audio encoding | YAMNet (Google) + OpenL3 | Open-source, interpretable, no licensing issues |
| Text encoding | SentenceTransformers (HF) | Lightweight, finetunable |
| Modeling | Hugging Face + PyTorch | Fine-tune small transformer for motif suggestions |
| API | FastAPI + Uvicorn | Fast, async, easy to audit |
| Frontend | React + Tailwind + Vite | Simple, responsive, fast |
| Auth/Consent | OAuth + signed forms (JSONWebSignature or blockchain hash) | Proof-of-consent; non-repudiable |
| Logging | W&B or MLflow | Experiment tracking + auditability |
| Hosting | DigitalOcean App Platform or AWS EC2 | Cost-controlled; user data stays within our VPC |

### Privacy & Auditability

- **No public API keys**: Model access restricted to signed consent holders.
- **Audit logs**: Every query, every generation, every consent action logged with timestamp + user + source tracks.
- **Encrypted consent records**: Artist/fan data encrypted at rest; only accessible to consent admin + advisory board.
- **Model weights: private, gated**: Fine-tuned model weights not published; only inference API is public.
