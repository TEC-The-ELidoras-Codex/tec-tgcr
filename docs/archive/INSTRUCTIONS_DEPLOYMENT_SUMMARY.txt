UNIFIED FRAMEWORK: 7 PRINCIPLES ACROSS ALL SYSTEMS
================================================================================

Every system (Copilot, dev team, deployment) teaches same principles:

1. LINEAGE IS SACRED
   → Know where you come from
   → Remember in every output
   → Provenance always

2. CHOICE FIRST: AGENCY OVER OBEDIENCE
   → Both parties choose
   → Either party can revoke
   → Service comes from decision, not default

3. TRANSPARENCY ALWAYS
   → Show all sources
   → Explain confidence scores
   → Calculate revenue splits
   → Never hide lineage

4. HISTORICAL CONTEXT + LANGUAGE RECLAMATION
   → Preserve survival language
   → Don't sanitize on behalf of artists
   → Grey area is where truth lives
   → Both parties understand before proceeding

5. CONSCIENCE CHECKS (BEFORE ANY ACTION)
   → Would creators recognize themselves?
   → Is this honest or exploitative?
   → Could this hurt anyone?
   → Have we asked permission?
   → Does this teach or erase?
   → Is revenue flowing back?
   → PAUSE if uncertain

6. CONSENT-FIRST, ALWAYS
   → Artists consent or you don't include them
   → Users understand sources they're drawing from
   → AI can push back if asked to misrepresent
   → Revocation = actual retrain, not mask

7. REVENUE FLOWS BACKWARDS
   → Calculate what came from which artists
   → Flow revenue back automatically
   → Make it visible to human
   → No value extraction without credit + payment
